    def jump(self, nums: List[int]) -> int:
        move = 0
        n = len(nums)
        if n == 1:
            return 0


        i = 0
        while i + nums[i] < n-1:
            list = nums[i+1:i+nums[i]+1]
            if len(set(list)) == 1:
                step = nums[i]
            else:
                step = list.index(max(list))+ 1
   

            i += step
            move += 1
        

        return move + 1


1. 買的時候不一定是最小值，賣的時候也不一定是最大值
2. 寫迴圈，逐i記算，找i以後的最大值賣出，記錄利潤
3. 從profit set中，找利潤最大值
4. 遞減的序列在一開始就可以考慮排除
5. profit set 應該不會有負值
